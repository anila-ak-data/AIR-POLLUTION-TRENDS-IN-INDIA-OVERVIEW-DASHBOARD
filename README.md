# AIR-POLLUTION-TRENDS-IN-INDIA-OVERVIEW-DASHBOARD
AIR POLLUTION TRENDS IN INDIA OVERVIEW DASHBOARD
This mini project analyzes the Air Pollution Trends in India using AQI values and relationship between vehicle registrations and air pollution levels across different states in India using Power BI. The project combines Air Quality Index (AQI) data and State-wise vehicle registration data to identify trends, comparisons, and possible correlations between transportation growth and environmental pollution.
